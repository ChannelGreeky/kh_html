@ 테이블 태그
-> 웹 문서에서 자료를 정리할 때 자주 사용
-> 행과 열로 이루어져 있고, 행과 열이 만나는 지점을 셀이라고 한다.

1. <tr></tr> : 한 개의 행(row)를 만드는 태그
2. <td></td> : 한 개의 열(col)을 만드는 태그
3. <th></th> : 열에 대한 제목을 표시하는 태그, 중앙 정렬 및 굴게 표시된다.
작성법
	<tr>
		<td>홍길동</td>
		<td>20살</td>
		<td>경기도</td>
	</tr>

4. <table> </table> : tr,td,th 를 하나의 테이블로 묶을 때 사용

border 속성 : 표의 테두리 두께를 지정할 수 있음

작성법
<table border = "두께"> </table>
	-> 두께 값은 숫자로 표시하고 클수록 두께가 커짐

5. rowspan 과 colspan
-> rowspan : <td> 태그의 속성으로 지정한 행만큼 해을 병합
-> colspan : <td> 태그의 속성으로 지정한 열만큼 열을 병합

작성법
<td rowspan = "숫자"> </td>
<td colspan = "숫자"> </td>

@ 그외 관련 태그
-> 테이블을 각각 나누어서 구조를 만들 수 있음

1. <thead></thead>
-> 테이블 구조를 나누는 태그, 테이블 당 1개만 가능

2. <tbody></tbody>
-> 테이블의 몸체를 표현하고 여러개 존재 가능

3. <tfoot></tfoot>
-> 테이블의 마지막을 표현하며 <thead>보다 뒤에 있어야 하고, 하나만 존재
