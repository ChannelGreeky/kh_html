@ h 태그
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>
-> 제목을 입력할 때 사용하는 태그, 폰트의 크기가 태그에 따라 정해져 있음
-> h태그 뒤 숫자로 구분

@ br 태그
-> 문장을 줄 바꾸기 할 때 사용

@ hr 태그
-> 페이지에 가로선을 만들어 줄 때 사용

@ p 태그
-> 한 개의 단락을 만들 때 사용
-> 자동으로 문장을 개행한 후 공백을 만들어 줌

@ pre 태그
-> 입력한 그대로 출력할 때 사용
-> 띄어쓰기, 들여쓰기, 줄 바꿈이 입력된 그대로 출력된다.

@ strong 태그 또는 b태그
-> 문장에서 문자를 강조하여 굵게 표시할 경우에 사용

@ em 또는 i 태그
-> 문자에 기울임을 주는 태그

@ blockquote 태그
-> 다른 블로그나 사이트의 글을 인용할 경우 사용
-> 자동 들여쓰기가 되어 다른 텍스트들과 구별할 수 있음
ex) <blockquote> 인용 내용 </blockquote>

@ q 태그
-> 다른 블로그나 사이트의 글을 인용할 경우 사용
-> 인용 구문에 " " 표시가 됨
ex) <q>인용 내용</q>

@ mark 태그
-> 배경 부분이 노란색이 되어 형광펜 표시가 된듯하게 출력
ex) <mark> 강조문구 </mark>

@ 그 외
1. <u></u> : 해당 글자에 밑줄 표시
2. <small></small> : 원 문자보다 작은 글씨
3. <sub></sub> : 아래 첨자
4. <sup></sup> : 윗 첨자
5. <cite></cite> 웹 문서나 포스트에서 참고할 때 사용 
	- 기울어져 있는 글씨 형태로 표시
