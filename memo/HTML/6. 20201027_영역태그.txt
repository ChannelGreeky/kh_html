@ iframe 태그
-> 웹 문서 안에 다른 웹 페이지를 추가 하는 태그
-> <iframe> ~ </iframe>
-> 현재는 보안 때문에 iframe으로 페이지를 호출할경우 사용이 안될 수 있음

속성
width/height 	: 페이지의 크기 설정
name		: 프레임의 이름
src		: 페이지의 경로

@ <div> 태그
-> block 형식의 공간을 분할
-> 공간 단위
-> 혼자서는 쓰이지 않으며, css와 함께 공간을 분할 할 때 사용 됨

@ <span> 태그
-> inline 형식의 공간을 분할
-> text 단위
