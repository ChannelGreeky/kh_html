@ 레이아웃 명명 규칙

wrapper			: 페이지 전체 영역
header			: 사이트 상단
container / content	: 본문 영역
footer			: 하단 footer 영역
navXXX			: 네이게이션 요소
	- GNB(Global Navigation Bar) : 사이트 최상위 전체 공통 네비게이션
	(메인 메뉴, 대분류 메뉴, 전체 사이트에서 사용되는 공통 메뉴바)
	- LNB(Local Navigation Bar) : 현재 서비스 영역만 해당되는 네비게이션(서브 메뉴)
	- SNB(Side Navigation Bar) : 페이지의 양 옆에 있는 사이드 네비게이션
	서브메뉴를 제외한 사이드 메뉴바
	- FNB(Foot Navigation Bar) : 하단 메뉴에 있는 네비게이션 하단 메뉴, 하단 로고, 주소 등
